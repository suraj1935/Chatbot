 Medical Encyclopedia Chatbot
An AI-powered medical chatbot that answers health-related questions using the Gale Medical Encyclopedia and Perplexity AI.

 Features
RAG System: Retrieval Augmented Generation with medical content
Medical Knowledge: 5157 text chunks from medical encyclopedia
AI-Powered: Uses Perplexity AI for accurate medical responses
Source References: Shows medical sources for each answer
 Usage
Ask any medical question in the chat interface
Get AI-powered answers based on medical encyclopedia
View source references for verification
 Technical Details
Framework: Streamlit
Embeddings: Sentence Transformers (all-MiniLM-L6-v2)
LLM: Perplexity AI
Vector Search: Cosine similarity
